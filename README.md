# Скрипт для автоматизированной настройки VPS
Честно написанный через нейросеть для упрощения настройки VPS. Работает на **Debian** и **Ubuntu**.

Скрипт помогает быстро подготовить новый сервер: обновить систему, создать пользователя, настроить SSH, включить UFW, Fail2ban и BBR.

## Что делает скрипт

* обновляет систему;
* создаёт нового пользователя;
* предлагает выбрать способ входа:

  * по паролю;
  * только по SSH-ключу;
* добавляет публичный SSH-ключ;
* отключает вход под `root` по паролю;
* меняет SSH-порт на случайный;
* показывает новый SSH-порт;
* устанавливает `btop`;
* устанавливает и настраивает `ufw`;
* включает rate limit для SSH;
* устанавливает и настраивает `fail2ban`;
* запрашивает один или несколько доверенных IP/CIDR;
* добавляет эти IP в whitelist UFW и `ignoreip` Fail2ban;
* открывает только активные порты и закрывает остальные;
* проверяет BBR и включает его, если он выключен.

## Поддерживаемые системы

* Debian
* Ubuntu

## Быстрый запуск

Если файл в репозитории называется `vps-bootstrap.sh`, запускай так:

```bash
bash <(wget -qO- https://raw.githubusercontent.com/nex77qoz/VPS-Setup/main/vps-bootstrap-universal.sh)
```

Или через `curl`:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/nex77qoz/VPS-Setup/main/vps-bootstrap-universal.sh)
```

## Как использовать

1. Подключись к серверу под `root`.
2. Выполни одну из команд выше.
3. Ответь на вопросы скрипта.
4. После завершения **обязательно проверь вход по новому SSH-порту в отдельной сессии**.

## Важные замечания

* Скрипт рассчитан на запуск от `root`.
* Если выбран вход только по SSH-ключу, пароль для нового пользователя не запрашивается.
* Если включён UFW, скрипт старается автоматически открыть только нужные порты.
* Доверенные IP добавляются в whitelist, чтобы Fail2ban не забанил тебя по ошибке.
* Не закрывай текущую SSH-сессию, пока не убедишься, что вход на новый порт работает.
